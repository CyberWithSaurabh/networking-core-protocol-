# Networking Core Protocols - Tryhackme

## 📌 Lab Overview

The **Networking Core Protocols** lab from TryHackMe introduces the fundamental protocols that enable communication between devices on a network.
This lab focuses on understanding how essential networking protocols operate and how they support
reliable data transmission across the internet.

Learners explore important protocols such as TCP, UDP, IP, DNS, and DHCP, and understand their role in sending, 
receiving, and managing network traffic.

---

## 🎯 Objectives

* Understand the purpose of core networking protocols
* Learn how devices communicate across networks
* Identify how IP addressing and domain name resolution work
* Explore how different protocols interact in real-world networking

---

## 🧠 Key Concepts Covered

* TCP (Transmission Control Protocol)
* UDP (User Datagram Protocol)
* IP (Internet Protocol)
* DNS (Domain Name System)
* DHCP (Dynamic Host Configuration Protocol)
